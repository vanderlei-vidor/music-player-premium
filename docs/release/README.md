# Release Docs

Historico de checklists, notas e revisoes de release.

## Arquivos versionados

- `RELEASE_CHECKLIST_*.md`: checklists especificos por versao.
- `RELEASE_NOTES_*.md`: notas preparadas para publicacao.
- `RELEASE_REVIEW_*.md`: revisoes e retrospectivas de release.

O checklist operacional mais recente continua na raiz em `RELEASE_CHECKLIST.md` para ficar facil de encontrar durante o ciclo atual.
